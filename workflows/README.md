# ワークフロー / Workflows

**ステータス / Status:** `Draft`

ワークフローは、複数の知識や成果物をつなぐ横断的な実行プロセスです。Knowledge は判断の根拠、Prompt は単一タスクの指示、Workflow は順序だった進め方、Skill は支援資産を含む再利用可能な能力、Agent はそれらを目的に応じて実行する主体です。

Workflows are cross-module execution processes that connect knowledge and deliverables. Knowledge supplies rationale; a Prompt instructs one task; a Workflow provides ordered execution; a Skill is a reusable capability with supporting assets; and an Agent selects and executes them toward an objective.
