# デザイン判断 / Design Decisions

**ステータス / Status:** `Draft`

この領域は、特定の設計課題に対する判断、その根拠、代替案、影響を記録します。原則は一般的な判断基準を、decision は個別の結論を扱います。新しい記録は [_template.md](./_template.md) から始め、必要になった時点で `DD-001-short-title.md` のように採番します。

This area records decisions for specific design questions, their rationale, alternatives, and impact. Principles provide general criteria; decisions record particular conclusions. Start new records from [_template.md](./_template.md) and number them, for example `DD-001-short-title.md`, when needed.
