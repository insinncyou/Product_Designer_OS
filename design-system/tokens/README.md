# デザイントークン / Design Tokens

**ステータス / Status:** `Draft`

## 目的 / Purpose

色、タイポグラフィ、余白、サイズ、モーションなどのデザイン判断を、再利用可能な名前とルールとして整理します。

Organize decisions such as color, typography, spacing, size, and motion into reusable names and rules.
