# コンポーネント / Components

**ステータス / Status:** `Draft`

## 目的 / Purpose

再利用可能な UI コンポーネントの役割、構造、状態、振る舞い、利用上の注意を記録します。

Document the role, anatomy, states, behavior, and usage guidance of reusable UI components.

## 構成 / Structure

- [Component inventory](./component-inventory.md) — 現在の対象と成熟度 / current scope and maturity
- [Component template](./_template.md) — 新規仕様の起点 / starting point for new specifications
- カテゴリー / categories: `actions/`, `forms/`, `navigation/`, `feedback/`, `data-display/`

各コンポーネントは、関連する [tokens](../tokens/README.md) と [patterns](../patterns/README.md) にリンクします。カテゴリーの空フォルダーは、承認済みのコンポーネント仕様をまだ含みません。

Each component links to related [tokens](../tokens/README.md) and [patterns](../patterns/README.md). Empty category directories do not yet contain approved component specifications.
