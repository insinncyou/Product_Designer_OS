# ガバナンスモデル / Governance Model

**ステータス / Status:** `Draft`

ガバナンスは、共有資産の判断責任、変更の可視性、利用者からのフィードバック経路を定義します。現時点では特定の組織ロールを仮定しません。資産ごとに提案者、レビュー責任者、承認者、保守者を明記することを将来の最低条件とします。

Governance defines decision accountability, change visibility, and feedback paths for shared assets. It does not assume specific organizational roles at this stage. A future minimum is to name proposer, reviewer, approver, and maintainer for each asset.
